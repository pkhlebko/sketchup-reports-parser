What is it

Sketchup Parser is a tool which extends Sketchup export csv file with fields from object defenition.

How to use

  1. Upload csv report from Sketchup.
  2. Select suitable template.
  3. Click `Trunsform`.
  4. Click `Download`.
  5. Enjoy.

Expected formats of Sketchup object definition

Furniture

Template
  1. [PartName (any symbols except space)]
  2. [Width (decimal)][separator(x or *)][Height (decimal)]
  3. [Edge bending flags width1, width2, height1, heigth2 (0 or 1 ex 1100)]
  4. [Comment (any symbols, optional)]
Example

Table_top  1600*700  1111  All corners rounded with R30

Frame building

Template
  1. [PartName (any symbols except space)]
  2. [Width (decimal)][separator(x or *)][Height (decimal, optional)]
  3. [Comment (any symbols, optional)]
Example

2by6  2000  King stud
