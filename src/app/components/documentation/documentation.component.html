Expected formats of Sketchup object definition

Furniture

  1. [PartName (any symbols except space)]
  2. [Width (decimal)][separator(x or *)][Height (decimal)]
  3. [Edge bending flags width1, width2, height1, heigth2 (0 or 1 ex 1100)]
  4. [Comment (any symbols, optional)]
